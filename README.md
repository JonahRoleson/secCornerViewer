# secCornerViewer
---
## Idea:
Click on the given map containing the section corner grid of grids.  
The Result is a description or multiple plausible descriptions of what that point is.  
If hovering over the plausible ones, it will show you the highlighted boxes of which it coordinates to.
Be able to select direction and corner/quarter and build out the text version into creating the point on the graph.
So it can go both ways.
List out the descriptions in order from smallest to largest.
Each hovered over description shows a bounding box of the smallest quarter it operates from.
Eventually add OCR and maybe a small AI to read the OCR into a usable format.
